# Rock Paper Scissors (Python)

A simple Rock Paper Scissors game written in Python that runs in the terminal.
The player competes against the computer, which randomly selects its move.

## Features

* Player vs Computer gameplay
* Random computer choices
* Input validation
* Option to quit the game anytime

## How It Works

1. The player enters **rock**, **paper**, or **scissors**.
2. The computer randomly selects one of the three options.
3. The program compares both choices and displays the winner.

## Requirements

* Python 3.x
* A code editor such as Visual Studio Code

## How to Run

1. Download or clone the project.
2. Open the project folder in Visual Studio Code.
3. Run the Python file:

```bash
python rock_paper_scissors.py
```

4. Enter your choice when prompted.

Example input:

```
rock
paper
scissors
quit
```

## Example Gameplay

```
Rock Paper Scissors Game
Choose rock, paper, or scissors (or quit): rock
Computer chose: scissors
You win!
```
